# How to Label the Tennis Ball 🎾

Thanks for helping! You're teaching the computer to find the tennis ball in videos.
It's easy: you just **click on the ball** in each picture.

## Start it up

1. Find the file called **`Label Ball`** (it has gears on the icon).
2. **Double-click it.**
3. A menu opens with big buttons. Pick one:
   - **Label a new video** — mark the ball in a fresh video.
   - **Fix the computer's guesses** — check the computer's work and fix mistakes.

A box pops up so you can choose the video file. Pick it and you're labeling!

## Labeling a new video

You'll see the video one picture at a time. For each picture:

- **Click right on the ball.** It marks the spot and jumps to the next picture.
- **Can't see the ball?** (It's hidden or flew off the screen.) Press **`N`**.
- **Went too fast / made a mistake?** Press **`A`** to go back one picture.
- **A little off?** Use the **arrow keys** to nudge the mark one tiny step.

There's a **zoomed-in circle in the top-right corner** so you can see the ball up
close and click exactly on it.

## Fixing the computer's guesses

The computer already guessed where the ball is. The guess shows up **YELLOW**.

- If the guess is **right**, press **`C`** to keep it. It turns **RED** (= checked).
- If the guess is **wrong**, just **click the correct spot** to fix it.
- Press **`U`** to jump to the next picture that still needs checking.

## When you're done

- Your work **saves by itself** — you can't lose it.
- Press **`Q`** any time to save and stop. (You can come back later and it picks up
  right where you left off.)
- Stuck? Press **`M`** for the menu, or click **"How do I use this?"** on the
  first screen.

## The buttons, all in one place

| Press this | What it does |
|-----------|--------------|
| **Click the ball** | Mark it and go to the next picture |
| **`N`** | The ball is not visible (hidden / off-screen) |
| **`A`** | Go back one picture |
| **arrow keys** | Move the mark one tiny step |
| **`C`** | (Fixing mode) The guess is right — keep it |
| **`U`** | Jump to the next picture that needs you |
| **`M`** | Open the menu / more help |
| **`Q`** | Save and stop |

That's it. Have fun, and thank you! 🎾
